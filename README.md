# Material & Meaning

How material conditions and meaning-making couple to shape health.

Material & Meaning is a public health research institute. It studies the coupling between material conditions (rent, wages, incarceration, psychiatric holds) and meaning-making (belief, practice, the secular position) as determinants of health, read through complexity theory and game theory.

This repository holds the institute's website. The site is static HTML and CSS with no build step, served by GitHub Pages from the repository root.

Live site: https://risaac09.github.io/material-and-meaning/
