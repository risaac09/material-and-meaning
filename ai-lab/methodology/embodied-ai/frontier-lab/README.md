# The Embodied AI Frontier Lab

The welcome container. If you are arriving here for the first time, start with
`the-story.md`, then read your part in `grader-invitation.md`.

This is Rubinstein Productions' one-person frontier lab. It gave an AI a body
map and built the governance to test it honestly. It mirrors how the big labs
govern, sized down to a studio, and it names its own limits in the open.

## What is here

- [`the-story.md`](the-story.md) — the narrative. What we built, why, and the
  one question the whole thing exists to answer.
- [`grader-invitation.md`](grader-invitation.md) — the invitation to the
  independent grader seat, the one rule that defines it, and how to accept.
- [`grader-runbook.md`](grader-runbook.md) — the exact steps to grade: run the
  wager, run the live probes, score blind, record the result.
- [`deck.md`](deck.md) — a slide-by-slide deck for the first welcoming. Source
  form, ready to render into Canva or Keynote.
